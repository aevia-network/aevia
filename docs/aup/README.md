# Acceptable Use Policy (AUP)

The Aevia AUP defines categories of content that the protocol MUST NOT actively distribute or incentivize, consistent with the axiom "persistence does not imply distribution."

## Status

Sprint 1 — drafting.

## Scope (preview)

- **Hard exclusions**: explicit sexual content, sex work, any sexualization of minors (absolute zero tolerance with NCMEC reporting), celebratory apologia of abortion, occultism/satanism/witchcraft as practice, drug apologia, actionable hate speech, violence apologia.
- **Gray zones**: to be reviewed with moderation council.
- **Client templates**: Family / Standard / Ministry ranking templates allow users to choose content curation within the protocol's policy bounds.

Full AUP v1 ships as Annex A of `docs/protocol-spec/8-moderation.md`.
